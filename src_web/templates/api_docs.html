API Documentation

Semua endpoint API mengembalikan response dalam format JSON dengan struktur yang konsisten.
Response Structure
Success Response:
{
  "success": true,
  "message": "Deskripsi pesan",
  "data": { ... },
  "timestamp": "2025-10-17 12:00:00"
}
Error Response:
{
  "success": false,
  "message": "Deskripsi error",
  "error": "Detail error",
  "timestamp": "2025-10-17 12:00:00"
}

Available Endpoints

GET /api/status
Try it

Mendapatkan status server dan informasi sistem.

Response Example:
{
  "success": true,
  "message": "Server running",
  "data": {
    "status": "online",
    "server_time": "17 October 2025, 12:00:00",
    "os_info": "Windows 10",
    "python_version": "3.10.0",
    "hostname": "DESKTOP-PC",
    "local_ip": "192.168.1.100",
    "port": 8080
  },
  "timestamp": "2025-10-17 12:00:00"
}

Try it

      "port": 8080,
      "status": "running"
    },

      "hostname": "DESKTOP-PC"
    },
    "folder": {
      "default": "C:/Data/Arsip"
    }
  },
  "timestamp": "2025-10-17 12:00:00"
}
